1. Si vous n’avez pas de compte sur SendInblue. Aller à cette adresse :

2. Puis, inscrivez-vous sur le site.

3. Une fois inscrit, vous devez vous rendre sur votre boîte mail, afin de valider votre inscription.

4. Il vous faut alors, terminer votre inscription.

5. Vous devez ensuite choisir le forfait que vous souhaitez utiliser. Tout dépend du nombre de mails que vous comptez envoyer pour la newsletter. Nous pensons que la version gratuite est suffisante pour votre village.

6. Vous voilà donc sur votre tableau de bord, c’est ici que vous pourrez trouver tout ce dont vous avez besoin pour créer et programmer votre newsletter.

7. Vous vous retrouvez sur votre tableau de bord. Depuis celui-ci, cliquer sur votre profil en haut à droite de votre page afin de dérouler ce menu. Une fois le menu accessible, cliquez sur “SMTP & API”.

8. Vous arrivez alors sur une page qui vous donne accès à une clé permettant l’utilisation du plugin SendInBlue sur Wordpress. Cette clé, il faudra la donner à l’un des admins de votre site, afin qu’il puisse relier le plugin de votre site et le site que vous allez utiliser dans la suite de ce tutoriel.

9. Pour utiliser SendInBlue sur votre site vous devez récupérer la clé API vue précédemment. Une fois cette clé récupérer, rendez-vous dans la partie admin de votre site Wordpress. Puis, rendez-vous dans le plugin SendInBlue présent dans le menu sur votre gauche.

10. Vous vous trouvez donc dans les réglages du plugin SendInBlue installé sur votre site. Il vous propose en première étape de vous créer un compte, cependant vous en avez déjà un. Vous vous intéressez donc à la seconde étape qui est l’utilisation de la Clé API que vous avez récupéré auparavant. Vous renseignez votre Clé API dans l’encadré et vous cliquez sur le bouton “Vous connectez”.

11. Une fois connecté, vous devez voir ce résultat.

12. Si vous avez ce résultat, c’est que tout est bon pour cette partie votre site est connecté à votre compte SendInBlue.

13. Vous venez donc de créer votre compte, et vous venez de voir comment relier votre site internet et votre compte SendInBlue. Maintenant, il faut activer les emails transactionnels. Pour cela, il vous faut envoyer un mail à cette adresse mail : contact@sendinblue.com. Voici un exemple, l’image suivante est le mail que j’ai envoyé pour activer mon compte.

14. Une fois le mail envoyé, rendez-vous sur le plugin SendInBlue sur votre site Wordpress.

15. Puis, cliquez sur “Formulaires” afin d’accéder au formulaire d’inscription de la Newsletter.

16. Il va alors vous falloir éditer le formulaire présent, et vous allez arriver sur une page ou vous pouvez personnaliser votre formulaire. Ce qui n’est pas forcément nécessaire hormis pour traduire certains termes.

17. Puis vous allez descendre cette page jusqu’à la section “Processus d’inscription” où vous allez quelle est la liste liée à votre formulaire d’inscription. Retenez le nom de cette liste, vous allez en avoir besoin pour plus tard.

18. Retourner sur votre tableau de bord sur le site de SendInBlue. Si vous êtes sur Wordpress, j’espère que vous avez gardé votre page ouverte sinon il va falloir la rouvrir. Et si vous étiez sur la page, où vous avez récupéré votre clé API, vous pouvez cliquer sur le logo de SendInBlue qui se situe en haut à gauche de la page.

19. Une fois revenu sur votre tableau de bord, cliquez sur Contacts. Vous arrivez alors, sur une page nommée Contacts. C’est ici que vous allez pouvoir voir toutes les personnes qui s’inscrivent sur la Newsletter de votre site internet.

20. Si vous avez bonne mémoire, vous pourrez facilement retrouver la liste reliée à votre formulaire d’inscription. Alors, pour cela il faudra vous rendre sur l’onglet Listes.

21. Vous pourrez alors retrouver la liste que vous aviez précédemment découverte, pour ce tutoriel le nom de celle-ci est “Votre première liste”. Une fois que vous avez trouvé la bonne liste il vous faut la renommer afin de bien la retrouver pour la suite. Pour cela cliquer sur le bouton “Actions” puis sur “Renommer” et vous pouvez alors la renommer comme vous le souhaitez.

22. Vous connaissez à présent le nom de la liste, où, les personnes qui s’inscrivent à la Newsletter présente sur votre site arrivent. Pour la suite, il va vous falloir retourner sur votre tableau de bord, vous allez créer votre Campagne d’email.

23. Une fois revenu sur votre tableau de bord, cliquez sur Email qui se situe sur la gauche de votre page. Vous arrivez sur une page nommée Campagnes email. C’est ici que vous allez pouvoir créer et programmer votre/vos campagnes d’emails. Cliquez sur “Créer ma première campagne Email”.

24. Vous pouvez alors vous lancer dans la création de votre newsletter. Rester en version classique, vous n’avez pas l’utilité de choisir A/B Test. Vous avez juste besoin d’un envoi de mail classique. Il ne vous reste plus qu'à compléter les informations de la première étape. Il est nécessaire de remplir le “Nom de la campagne” pour poursuivre la création de votre newsletter. L’objet et le texte d’aperçu quant à eux ne sont pas obligatoires. Vous pouvez modifier l’email et le nom de l’expéditeur si vous le souhaitez, mais si vous avez correctement rempli votre profil lors de l’inscription, pas besoin d’y toucher.

25. Une fois que vous avez tout complété, vous pouvez vous rendre sur la prochaine étape que vous pouvez trouver en haut à droite de la page.

26. Une fois sur cette page, vous allez devoir choisir par quel moyen vous allez créer votre Newsletter. Vous avez donc le choix entre ces 3 propositions. Je vous conseille l’éditeur Drag & Drop, c’est un outil pratique, qui vous permet de créer des mails de bonne qualité qui sont agréables à lire et facilement faisables. Cliquez sur l’éditeur Drag & Drop, c’est le moyen le plus pratique de créer un email.

27. Vous venez d’arriver sur la page qui va vous permettre de créer de A à Z votre email. Ce que vous voyez sur la gauche de votre écran, est une liste d’éléments que vous pouvez prendre et déposer sur la partie centrale de votre page. Au centre de votre page, vous pouvez voir que des éléments sont déjà présents. Vous n’avez donc, plus qu’à créer le mail que vous souhaitez en retirant ou en ajoutant les éléments que vous souhaitez. Une fois que vous avez fini de créer votre mail, vous n’avez plus qu’à cliquer sur “Enregistrer & Cliquer”.

28. Une fois que vous aurez enregistré, vous allez voir ceci. C’est un aperçu du travail que vous venez de quitter, vous pouvez même avoir un aperçu mobile de ce que rend votre travail. Si vous êtes satisfait du rendu vous pouvez passer à la prochaine étape et si vous souhaitez modifier le contenu de l’email vous n’avez qu’à retravailler votre email.

29. La prochaine étape consiste à sélectionner la liste de contacts que vous avez définis auparavant. Une fois sélectionnée, vous pouvez passer à la prochaine étape.

30. Vous arrivez sur une page qui vous présente un récapitulatif de tout ce que vous venez de créer. Il ne vous reste plus qu’à cliquer sur le bouton “Programmer”.

31. Ici, vous pouvez choisir soit d’envoyer le mail dès maintenant, soit de programmer son envoi. Il ne vous reste donc qu’à choisir, et le mail s’enverra à tous les contacts que vous avez définis, au jour et à l’heure que vous avez définis.

32. Voilà vous venez de créer et programmer votre Newsletter.
