1. Lorsque vous êtes dans le tableau de bord de votre site, rendez vous sur l’onglet article qui se trouve dans la barre à gauche de votre écran.

2. Vous atterrissez sur la liste de tous vos articles. Quand ils sont trop nombreux, ils peuvent être rangés sur plusieurs pages. Vous pouvez les consulter en cliquant sur les flèches à droite de votre écran.

3. Cliquez sur le bouton « ajouter » tout en haut de votre écran, à côté du titre « Articles ».

4. Une nouvelle page s’ouvre. Vous devez tout d’abord entrer le titre de votre article. Dans cet exemple, disons que l’article s’appelle « Couvre-feu obligatoire entre 20h et 6h ».

5. Il est maintenant temps de créer le contenu de l’article. Pour cela vous devez cliquer sur le bouton “modifier avec Elementor”.

Il y a plusieurs manières d’écrire du contenu sur wordpress. Nous avons décidé d’utiliser le plug-in Elementor qui facilite grandement la mise en page de vos articles.

Pour créer le contenu sur Elementor, un tutoriel complet est dédié à ce sujet, juste ici.

6. Quand vous avez terminé la mise en page, rendez-vous dans les réglages. Pour cela cliquez sur l’icône engrenage en haut à droite.

7. Un menu va s’ouvrir à droite de votre écran (il se peut qu’il soit déjà ouvert).

8. Dans ce menu, vous retrouverez les informations générales de votre article.

Dans visibilité, vous avez plusieurs options :

  • public = tout le monde peut le voir
  • privé = seules les personnes gérant le site web peuvent le voir
  • mot de passe = l’article est publié sur le site mais protégé par un mot de passe. Seules les personnes le possédant peuvent y accéder.

Dans publier : vous pouvez choisir de publier l’article immédiatement ou de programmer sa publication. Si vous ne touchez à rien, il sera automatiquement publié à l’instant même où vous confirmerez la publication.

Dans Yoast SEO, on retrouve un bilan du plugin Yoast qui vous indique si le référencement de votre article est correct (Si c’est le cas, tous les voyants sont au vert). Si vous ne l’avez pas fait ou si les voyants sont au rouge ou orange, ce tutoriel vous indique comment utiliser Yoast et bien référencer votre article.

Le permalien, c’est le lien de votre article. Vous pouvez le changer si vous estimez que le lien est trop long ou pas assez lisible.

Dans “catégories”, vous devez absolument cocher la case “Actualités”. L’article sera ainsi classé et apparaîtra dans les articles récents sur la page d’accueil et sur la page des actualités. Sinon, l’article ne sera pas visible des utilisateurs.

[ Point info : Le référencement, quesako ? Quand un article est bien référencé, il apparaît dans les premiers résultats du moteur de recherche, selon la requête tapée par l’utilisateur dans la barre de recherche. Dans notre cas, il s’agit d’un référencement naturel (on ne paie pas pour apparaître dans les premiers résultats). Pour permettre aux moteurs de recherche de mettre notre site dans les premiers résultats d’une recherche, il faut remplir certains critères, comme renseigner des mots-clés par exemple. ]

Étiquettes : Vous pouvez ajouter des mots-clés qui décrivent vos articles afin que les utilisateurs et les moteurs de recherche le retrouvent plus facilement sur votre site. Cela s’avère utile si un utilisateur recherche “lachapelle-sous-chaux couvre-feu” sur Google dans notre cas.

Ajoutez l’image qui sera mise en avant sur votre article. Il s'agit souvent de l’image que vous aurez mis en première place dans votre article, mais elle peut différer si vous le souhaitez. Celle-ci s’affichera sur la page d’accueil et d’actualité pour illustrer votre article. il est nécessaire que vous l’ajoutiez pour garder une belle mise en page.

Ajoutez un extrait à votre article. Vous pouvez reprendre les premières lignes de votre article ou en faire un résumé. Cet extrait apparaîtra sur la page d’accueil et la page d’actualité, il est nécessaire que vous le complétiez pour garder une belle mise en page.

Décochez les deux cases dans la section commentaire. Il n’y aura ainsi pas de commentaires sous les articles.

Finalement, vérifier que le modèle choisi dans Attributs d’articles est “modèle par défaut”. Cette mise en page est appropriée pour les articles car on retrouve toujours le widget “article récent” à gauche des articles.

9. Quand vous avez terminé la mise en page et que vous êtes prêt à publier l’article, cliquez sur “Mise à jour” en haut à gauche. Vous pouvez également enregistrer l’article dans les brouillons et reprendre plus tard. Il y a également la possibilité de prévisualiser votre article dans un nouvel onglet.

10. Voilà le résultat sur la page d’accueil et la page actualités.
